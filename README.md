# Frontend
## Install
- Nodejs required
- angular cli required

## How to run
### Dev mode
```bash
npm start
```
# Backend
- Run Spring boot in e.g. STS
- Command Line
## How to run command line
```bash
mvn clean install
cd backend
mvn spring-boot:run
```

# Build & Run
```bash
mvn package
java -jar backend/target/backend-0.1.0-SNAPSHOT.jar
```
